# 钱包余额查询 (getBalance)

## 功能

查询多链钱包地址的原生代币余额, 支持 EVM 链 (ethereum/polygon/bsc)、Bitcoin、Solana。

## 接口签名

```typescript
function getBalance(chain: ChainId, address: string, rpcUrl?: string): Promise<ToolResult<BalanceData>>
```

向后兼容: `getWalletBalance(address)` -> 内部调用 `getBalance('ethereum', address)`

## 参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| chain | ChainId | 是 | `'ethereum' \| 'polygon' \| 'bsc' \| 'bitcoin' \| 'solana'` |
| address | string | 是 | 钱包地址 |
| rpcUrl | string | 否 | 自定义 RPC 节点 URL |

## 返回值

```typescript
interface BalanceData {
  chain: string;
  address: string;
  balance: string;
  unit: string;       // ETH / MATIC / BNB / BTC / SOL
  decimals: number;
}
```

## 数据源

| 链 | 适配器 | 数据源 | 地址验证 |
|----|--------|--------|---------|
| ethereum/polygon/bsc | EvmChainAdapter | ethers.js JsonRpcProvider | ethers.isAddress() |
| bitcoin | BitcoinAdapter | blockchain.info API | Base58/Bech32 正则 |
| solana | SolanaAdapter | JSON-RPC API | Base58 公钥格式 |

EVM 默认 RPC: `eth.llamarpc.com` (可通过环境变量覆盖)

## 示例调用

```json
// 请求
{ "name": "getBalance", "arguments": { "chain": "ethereum", "address": "0x742d35Cc6634C0532925a3b844Bc9e7595f..." } }

// 成功响应
{ "success": true, "data": { "chain": "ethereum", "address": "0x...", "balance": "1.5234", "unit": "ETH", "decimals": 18 }, "source": "ethereum-rpc", "timestamp": "2025-01-01T00:00:00Z" }
```

## 错误处理

| 错误 | 返回 |
|------|------|
| 地址格式无效 | "无效的钱包地址" |
| 不支持的链 | "不支持的链" + 支持列表 |
| RPC 连接失败 | "获取余额失败" |
