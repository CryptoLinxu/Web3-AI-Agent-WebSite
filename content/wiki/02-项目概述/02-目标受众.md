# 目标受众

## 三类主要用户

### 1. Web3 前端工程师（转型 Agent 方向）

**背景**：熟悉前端框架与 Web3 生态，对 Agent 工程化尚不系统。

**学习路径**：
- 第 1 周：LLM API + Function Calling → 最小工具调用闭环
- 第 2 周：Agent Loop + Web3 工具 → 真实数据查询
- 第 3 周：Memory + RAG → 稳定系统 Prompt
- 第 4 周：工程化 + 部署 → 在线可访问版本

### 2. 转行软件开发者（跨领域入门）

**背景**：有传统开发经验，对 Agent 概念较陌生。

**学习路径**：
- DISCOVER + EXPLORE → 熟悉项目地图
- DEFINE → 需求澄清与边界定义
- DELIVER → 按流水线推进交付

### 3. AI Agent 学习者（进阶实践）

**背景**：了解 Agent 基本概念，需要工程化落地经验。

**学习路径**：
- 从 origin 进入相应流水线
- 严格遵循 define → check-in → design → build → closeout
- QA 执行 RED/GREEN，Coder 最多 10 轮自愈，Audit >=80 分通过

## 按层次适配

| 层次 | 策略 |
|------|------|
| 初学者 | DISCOVER/BOOTSTRAP + EXPLORE，配合按周学习清单 |
| 中级 | DEFINE 为核心，完成 pm/prd/req 后进入 DELIVER |
| 高级 | VERIFY/GOVERN 为主线，聚焦治理闭环 |
