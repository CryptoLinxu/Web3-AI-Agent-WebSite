# Init-Docs 文档初始化技能

**职责**：新项目首次建立文档体系，扫描项目骨架、构建初始地图与索引、生成基础结构化文档。

## 触发场景

- 新项目首次建立文档体系
- 历史文档迁移与结构重建
- 重建基础索引与初始文档结构

## 输入/输出

- **输入**：代码库现状、现有文档
- **输出**：初始地图、初始索引、基础结构化文档

## 执行流程

1. 扫描项目骨架
2. 建立第一版地图和索引
3. 生成初始文档结构
4. 交给 update-map

## 核心规则

| 规则 | 说明 |
|------|------|
| BOOTSTRAP 专用 | 任务类型为 BOOTSTRAP 时触发 |
| 不承担功能开发 | 仅负责文档基建 |
| 不代替后续 digest | 仅做初始化 |
| 初始化完成后交由正常 V3 链路 | 移交 update-map 继续演化 |
