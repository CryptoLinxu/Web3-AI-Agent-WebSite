# 主题系统

## 概述

基于 React Context + CSS变量的主题系统，支持深色/浅色/系统跟随三种模式，集成 RainbowKit 钱包组件主题适配，完全消除 SSR 闪烁。

## 主题模式

| 模式 | 值 | 说明 |
|------|---|------|
| 系统跟随 | `'system'` | 监听 `prefers-color-scheme` 媒体查询 |
| 浅色 | `'light'` | 固定浅色 |
| 深色 | `'dark'` | 固定深色 |

## 核心组件

**ThemeProvider**: 管理主题状态，操作DOM属性(`data-theme` + `dark`类名)，持久化到localStorage。

**ThemeContext**: 提供 `useTheme()` Hook -> `{ theme, setTheme, resolvedTheme }`

**ThemeSwitcher**: UI切换按钮，支持 `aria-pressed` 无障碍属性。

**RainbowKitProviderWrapper**: 处理 RainbowKit 主题配置，通过 `mounted` 状态避免水合警告。

## SSR闪烁消除机制

在 `layout.tsx` 中注入内联同步脚本(`getThemeInitScript()`)，在 React 执行前设置 `data-theme` 和 `dark` 类，配合 `suppressHydrationWarning` 消除水合警告。

## CSS变量表

```css
:root[data-theme="dark"] {
  --bg-primary: 13 13 13;
  --bg-secondary: 24 24 27;
  --bg-tertiary: 39 39 42;
  --text-primary: 250 250 250;
  --text-secondary: 161 161 170;
  --border-color: 63 63 70;
  --accent-primary: 124 58 237;      /* 紫色主题色 */
  --accent-secondary: 139 92 246;
}

:root[data-theme="light"] {
  --bg-primary: 255 255 255;
  --bg-secondary: 249 250 251;
  --bg-tertiary: 243 244 246;
  --text-primary: 17 24 39;
  --text-secondary: 107 114 128;
  --border-color: 229 231 235;
  --accent-primary: 124 58 237;
  --accent-secondary: 139 92 246;
}
```

使用方式: `text-[rgb(var(--text-primary))]`、`bg-[rgb(var(--bg-secondary))]`

## RainbowKit主题配置

```typescript
{
  accentColor: '#7C3AED',
  accentColorForeground: 'white',
  borderRadius: 'large',
  fontStack: 'system',
  overlayBlur: 'small'
}
```

## Tailwind集成

通过 `dark` 类名同步，兼容 Tailwind CSS 的 `darkMode: 'class'` 模式。CSS变量覆盖范围包括滚动条、代码块、光标动画、文本选择等全组件适配。
