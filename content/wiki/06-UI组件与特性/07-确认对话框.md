# 确认对话框组件

## 概述

可复用的模态确认对话框，用于处理危险操作（如删除对话）的二次确认。支持多种视觉变体、键盘导航和无障碍访问。

## 组件接口

```typescript
interface ConfirmDialogProps {
  isOpen: boolean              // 显示状态
  title: string                // 标题
  message: string              // 提示消息
  confirmText?: string         // 确认按钮文本（默认"确认"）
  cancelText?: string          // 取消按钮文本（默认"取消"）
  variant?: 'danger' | 'warning' | 'info'  // 视觉变体
  isLoading?: boolean          // 加载状态（禁用按钮）
  onConfirm: () => void        // 确认回调
  onCancel: () => void         // 取消回调
}
```

## 变体样式

| 变体 | 确认按钮样式 | 用途 |
|------|------------|------|
| danger | `bg-gradient-to-r from-red-500 to-rose-500` | 删除等不可逆操作 |
| warning | `bg-gradient-to-r from-amber-500 to-orange-500` | 需注意的操作 |
| info | `btn-primary`（紫色渐变） | 一般确认 |

## 交互特性

- ESC键快速关闭（useEffect绑定键盘事件）
- 点击遮罩层关闭
- Loading状态下禁用按钮 + 旋转动画
- 毛玻璃背景（`backdrop-blur-sm`）
- 淡入+缩放弹出动画

## 使用示例

```tsx
<ConfirmDialog
  isOpen={showDeleteDialog}
  title="删除对话"
  message="确定要删除此对话吗？此操作不可恢复。"
  variant="danger"
  confirmText="删除"
  isLoading={isDeleting}
  onConfirm={handleDeleteConfirm}
  onCancel={() => setShowDeleteDialog(false)}
/>
```
