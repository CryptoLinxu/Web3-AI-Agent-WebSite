# 设置面板组件

## 概述

侧滑设置面板，集成主题选择和内存策略配置功能，替代了独立的 ThemeSwitcher 组件。

## 组件接口

```typescript
interface SettingsPanelProps {
  isOpen: boolean
  onClose: () => void
  memoryStrategy: MemoryStrategy
  onMemoryStrategyChange: (strategy: MemoryStrategy) => void
}

type MemoryStrategy = 'l3-compression' | 'l2-sliding-window'
```

## 设置项

### 主题选择

| 选项 | 图标 | 实现 |
|------|-----|------|
| 系统跟随 | 🖥️ | `prefers-color-scheme` 媒体查询 |
| 浅色模式 | ☀️ | 固定 light |
| 深色模式 | 🌙 | 固定 dark |

通过 `useTheme()` Hook 与 ThemeProvider 交互，切换即时生效并持久化到 localStorage。

### 内存策略

| 策略 | 说明 | 适用场景 |
|------|------|---------|
| L3 摘要压缩 | AI生成摘要 + 保留最近消息，有额外API调用 | 复杂长对话 |
| L2 滑动窗口 | 纯本地截断，无额外调用 | 简单对话/性能敏感 |

切换后通过 `onMemoryStrategyChange` 回调通知父组件创建新的 MemoryManager 实例。

## UI设计

| 元素 | 样式 |
|------|------|
| 背景遮罩 | `fixed inset-0 bg-black/60 backdrop-blur-sm` |
| 面板容器 | `fixed right-0 top-0 h-full w-full max-w-md` |
| 主题选项布局 | `grid grid-cols-3 gap-2` |
| 展开/收起 | CSS过渡动画 |

支持点击遮罩层关闭、响应式布局适配移动端。
